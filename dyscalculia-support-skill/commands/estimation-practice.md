# Estimation Practice

Generate estimation exercises that build number sense and magnitude awareness.

## Process

1. Select the topic area and difficulty level
2. Generate the requested number of exercises using everyday contexts
3. Each exercise should:
   - Present a real-world scenario requiring estimation
   - Provide helpful benchmarks or reference points
   - Include the actual answer (hidden until the user estimates)
   - Explain the reasoning strategy for arriving at a good estimate

## Topic Areas

- **Money**: Estimating costs, totals, change, budgets
- **Time**: Estimating durations, arrival times, schedules
- **Distance**: Estimating lengths, heights, travel distances
- **Weight**: Estimating mass of everyday objects, ingredients
- **Quantity**: Estimating counts, amounts, populations

## Output Format

Return numbered exercises, each with:
- The estimation scenario
- Helpful benchmarks
- Space for the user's estimate
- The actual answer with explanation
- The estimation strategy used
