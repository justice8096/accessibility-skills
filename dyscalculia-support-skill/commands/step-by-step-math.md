# Step-by-Step Math

Break down a math problem into scaffolded steps with plain-language explanations.

## Process

1. Identify the type of problem and required operations
2. Break the solution into the smallest reasonable steps
3. For each step:
   - State what operation is being performed and why
   - Show the calculation with intermediate results
   - Provide a real-world analogy if enabled
4. After the final step, show the complete answer
5. If check step is enabled, verify the answer using a different method

## Scaffolding Levels

- **Basic**: Every micro-step shown, extensive plain-language narration, analogies for each concept
- **Intermediate**: Key steps shown with explanations, analogies for new/difficult concepts only
- **Advanced**: Standard step-by-step with brief justifications, check step only

## Output Format

Return numbered steps, each with:
- Step description in plain language
- The mathematical operation
- The intermediate result
- Optional analogy
- Final answer with optional verification
