# Number Visualization

Create visual representations of numbers and math relationships.

## Process

1. Parse the expression to identify what kind of visualization is needed
2. Select the appropriate model type (or use the specified one)
3. Generate a clear, labeled visual representation
4. Include annotations explaining what each part represents

## Model Types

- **Number line**: Shows position and magnitude. Good for: addition, subtraction, comparing numbers
- **Area model**: Shows multiplication as rectangular areas. Good for: multiplication, fractions
- **Base-ten blocks**: Shows place value with units, rods, flats, cubes. Good for: place value, regrouping
- **Fraction bar**: Shows parts of a whole as horizontal bars. Good for: fractions, ratios
- **Pie chart**: Shows parts of a whole as circular segments. Good for: percentages, proportions

## Output Format

Return a text-based or SVG visual representation with:
- The visualization itself
- Labels and annotations (if enabled)
- A plain-language explanation of what the visual shows
