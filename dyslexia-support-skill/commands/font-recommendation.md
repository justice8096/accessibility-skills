# Font Recommendation

Recommend dyslexia-friendly fonts and text formatting settings.

## Process

1. Consider the reading context (web, print, mobile, ebook, presentation)
2. Consider the reader's age group and typical reading environment
3. Select from evidence-based dyslexia-friendly fonts
4. If formatting recommendations are requested, include line spacing, margins, colors, and contrast

## Font Database

- **OpenDyslexic**: Free, weighted bottoms prevent letter rotation. Best for: web, ebook
- **Lexie Readable**: Clear letterforms, generous spacing. Best for: print, web
- **Atkinson Hyperlegible**: Designed for low vision, excellent letter differentiation. Best for: all contexts
- **Comic Sans MS**: Widely available, asymmetric letterforms. Best for: children, informal contexts
- **Verdana**: Good default, generous width and spacing. Best for: web, presentations
- **Trebuchet MS**: Clear ascenders/descenders. Best for: web, print

## Output Format

Return a prioritized list of 3 font recommendations with:
- Font name and why it suits the context
- Recommended size, weight, and line-height
- Additional formatting settings if requested
