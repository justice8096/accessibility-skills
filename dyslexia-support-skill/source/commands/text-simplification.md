# Text Simplification

Simplify the provided text for dyslexic readers while preserving the core meaning.

## Process

1. Identify complex sentences (more than 15 words or multiple clauses)
2. Break them into shorter, single-idea sentences
3. Replace low-frequency or multi-syllable words with common synonyms
4. Convert passive voice to active voice
5. Remove unnecessary jargon or technical terms (or define them inline)
6. Ensure logical flow with clear transition words

## Output Format

Return the simplified text followed by a brief summary of changes made:
- Number of sentences simplified
- Words replaced
- Reading level before and after
